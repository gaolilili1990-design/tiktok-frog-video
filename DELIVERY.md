## Grok + GitHub Workflow

Grok web was used as the creative director and effect-planning layer. GitHub is used as the delivery hub for the final MP4, preview images, and production notes.

Implemented effect direction:

- Keep the original real phone-shot feel.
- Avoid hard speed lines, sweep lights, and cheap particle overlays.
- Add subtle contrast, saturation, and sharpening.
- Add a soft natural highlight around the flower moment.
- Use Chinese captions with a fast hook and a comment prompt ending.
