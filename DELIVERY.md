# TikTok Frog Video Delivery

This repository stores the delivery package for the green frog desktop-object TikTok video project.

## Main outputs

- `frog-tiktok-v2.mp4`: final local TikTok-style edit, 9:16, 1080x1920, about 13 seconds.
- `frog-heygen-reference-clean.mp4`: clean reference version for HeyGen or future AI enhancement.
- `frog-tiktok-v2-poster.jpg`: poster preview.
- `frog-tiktok-v2-contact.jpg`: contact-sheet QA preview.

## Creative angle

Cute/healing desktop frog object on a pink background. TikTok packaging uses a strong first-second hook, Chinese bounce captions, flash-frame impact, subtle handheld motion, and a comment-bait ending.

## Playback

Open or download `frog-tiktok-v2.mp4` from this repository to play the final version.
